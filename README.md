# DevOps Health Check Scripts

Простой bash-скрипт для мониторинга основных показателей сервера: 
использование диска, памяти и время работы (uptime).

## Что делает

- Проверяет использование диска и выводит предупреждение, если превышен порог
- Проверяет использование памяти и выводит предупреждение при превышении порога
- Показывает аптайм сервера

## Использование

\`\`\`bash
chmod +x check_disk.sh
./check_disk.sh
\`\`\`

## Пример вывода

\`\`\`
Текущее использование диска: 46%
ВНИМАНИЕ: диск заполнен более чем на 20%!
Текущее использование памяти: 23%
Всё в порядке, места достаточно.
Аптайм сервера: up 1 hour, 58 minutes
Проверка завершена
\`\`\`

## Технологии

- Bash
- Linux (systemd, ss, df, free)
